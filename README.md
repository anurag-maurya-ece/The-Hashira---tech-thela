# 🛒 Tech Thela AI
### *Digitizing Street Vendors, Empowering Local Commerce*

[![Tech Stack](https://img.shields.io/badge/Stack-Next.js%20%7C%20MongoDB%20%7C%20Firebase-0070f3)](https://nextjs.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**Tech Thela AI** is a hyper-local street vendor aggregator and AI-powered prediction engine. We aim to formalize the unorganized street vendor sector while providing consumers with the convenience of an "Ola-style" interactive map for their daily needs.

---## 🚀 Key Features

### 👨‍🌾 For Vendors
- **Modernized AI Cart Detection**: Leveraging **TensorFlow.js (COCO-SSD)** for real-time item detection directly in the browser. Fast, accurate, and offline-ready!
- **hhh-Inspired Dashboard**: A premium, dark-themed interface designed for maximum efficiency and visual appeal.
- **Bulk Inventory Updates**: Seamlessly update your entire cart inventory with a single click after AI detection.
- **Manual Adjustments**: Hybrid workflow allowing quick manual tweaks to AI-detected items.

### 🛍️ For Consumers
- **Live Market Radar**: Real-time "Ola-style" interactive map to find the nearest vendors.
- **Premium User Experience**: Clean, modern interfaces for both login and browsing.
- **Multilingual Support**: Available in English, Hindi, and Hinglish.

---

## 🛠️ Tech Stack
- **Frontend**: Next.js (React), Tailwind CSS, TensorFlow.js
- **Backend**: Node.js, MongoDB Atlas
- **Auth**: Firebase Authentication (Phone & Password)
- **Deployment**: Vercel & Koyeb
